## APLIKASI PEMABAYARAN SPP
Aplikasi ini memiliki 3 hak akses level login, yang diantaranya :

Level Admin
* Login
* Logout
* CRUD data siswa
* CRUD data petugas
* CRUD data Kelas
* CRUD data SPP
* Entri Transaksi Pembayaran
* Lihat Histori Pembayaran
* Generate Laporan

Level Petugas
* Login
* Logout
* Entri Transaksi Pembayaran
* Lihat Histori Pembayaran

Level Siswa
* Login
* Logout
* Lihat Histori Pembayaran
